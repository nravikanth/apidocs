Welcome to Hotels API’s documentation!¶

Goibibo Hotel API implements RESTful web services allowing a user to search Hotels, Book and Cancel. All Server/Client exchanges proceed using standard JSON format. You should have a basic familiarity with software development and web-services to proceed with this documentation. Use the API in any language that supports Web-Services. You would need the API login credentials and URL to access this API.

Basic Auth mechanism is used for authorization, use the given goibibobusiness credentials to access the API’s.

Access to the production environment is available at:

URL: http://goibibobusiness.com/api/hotels/b2b/

You can follow the rest of the documentation on how to use the APIs. All the functionalities in this API are enabled, unless stated otherwise.
