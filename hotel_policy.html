Hotel Cancel PolicyÂ¶

  • Description:

    To get the Hotel policy and Hotel Cancel policy, required params are query string, hotel code (hc), Room type code (rtc), Room plan code (rpc), forward params (fwdp) and ibibo partner value (ibp)

    1. Hotel code (hc) can be used as used in Hotel Details API
    2. Room type code (rtc) is obtained from Hotel Details API under rooms_data
    3. Room plan code (rpc) is obtained from Hotel Details API under rooms_data
    4. Forward params is obtained from Hotel Details API under rooms_data
    5. Ibibo partner value is obtained from Hotel Details API under rooms_data
    6. Offercode value, which is obtained from Hotel Details API under rooms_data

Note

These validations hold for Hotel Policy API:

  • Checkout Date is after Checkin Date
  • No. of passengers cannot exceed 9
  • Minimum of 1 adult required for booking
  • No. of rooms cannot exceed 4
  • No of infants cannot exceed no. of adults
  • Valid hotel code
  • Room plan code (rpc) & Room type code (rtc)
  • Ibibo partner value (ibp) & Forward params (fwdp)

  • Sample request URL:

    Request Method : GET

    rtc, rpc and ibp values will be obtained from Hotel details API

    When fwdp value and offercode is null i.e.,:
